> > 1a.* - UHCI/EHCI

> > 1c.* - PCIe root ports
> > 1d.* - UHCI/EHCI
> > 1e.0 - PCI Bridge
> > 1f.0 - ISA Bridge
> > 1f.2,5 - SATA
